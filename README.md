# HomeBot
Kotlin Android app that enables you to remap the home button long press on Android devices.

This project is a fork of https://github.com/kklorenzotesta/homebot and integrates features from https://github.com/teja2495/quick-search to provide a fast, native search overlay.

# Usage
The app has to be set as your Assistant app in Settings > Apps > Advanced > Default apps > Assist & voice input. Once set, you can configure actions such as:
- **Quick Search**: A floating search bar for apps and web.
- **Toggle Brightness**: Quickly switch between brightness modes.
- **Toggle Flashlight**: Turn on/off your phone's torch.
- **Launch Apps/Shortcuts**: Open any app or system shortcut.

### Warning
Using HomeBot might disable features such as Active Edge squeeze to trigger Google Assistant on Pixel devices.

# Requirements
Android 6.0 or later (Some features like Quick Search require overlay permissions).
